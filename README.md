# PaisaBazaar Credit Risk & Financial Behaviour Analysis Dashboard

## Project Overview

This project analyzes customer credit behavior and financial behavior using Power BI. The dashboard helps identify factors affecting customer credit scores and provides actionable insights for risk management, loan approvals, and customer segmentation.

The analysis focuses on:

- Credit Score Distribution
- Customer Occupation Analysis
- Credit Utilization
- Outstanding Debt
- EMI Burden
- Payment Behavior
- Credit Mix
- Credit History
- Loan Analysis

---

## Problem Statement

Financial institutions here Paisabazaar need to identify high-risk customers before loan defaults occur. This dashboard provides a comprehensive view of customer credit behavior and highlights key risk indicators affecting credit scores.

---

## Dataset Information

The dataset contains customer financial and credit-related information.

### Features Used

| Category                           |  Columns |
|-----------------------------|-----------|
| Customer Details              | Age, Occupation |
| Income Information          | Annual_Income, Monthly_Inhand_Salary |
| Banking Details                 | Num_Bank_Accounts, Num_Credit_Card |
| Loan Information              | Num_of_Loan, Outstanding_Debt |
| Payment Behaviour           | Payment_Behaviour, Delay_from_due_date |
| Investment Information    | Amount_invested_monthly |
| Credit Metrics                   | Credit_Utilization_Ratio, Credit_Mix |
| Target Variable                  | Credit_Score |

## 🔍 Insights Generated

Some key business insights include:

- Customers with lower credit scores exhibit higher risk profiles.
- Income level significantly impacts creditworthiness.
- Certain customer segments contribute disproportionately to overall portfolio risk.
- Credit score and loan amount show strong correlation with approval decisions.
- Risk categories help prioritize monitoring and intervention strategies.
